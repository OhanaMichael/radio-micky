## ניתוח SEO — מיקום במנוע חיפוש (מאי 2026)

### 1. אביזורי רכב בטבריה

**Local Pack (3 ראשונים):** דודיק (4.9 / 110 ביקורות) | אבנר (4.6 / 67) | אוהד התקנות (4.9 / 8)
**רדיו מיקי:** לא מופיע ב-Local Pack. מופיע בתוצאות אורגניות דרך easy.co.il שמקשר אליו.
**סננת #1 (easy.co.il):** "אביזרים לרכב — הכי טוב בטבריה" (מדריך לרדיו מיקי בתוכו)
**פונקציות AI נוספת:** לא נצפת
**פועל נדרש: עוד ביקורות ב-Google Business Profile (49 ביקורות לעומת 110 לדודיק)**

### 2. מולטימדיה תואם מקור בטבריה

**Local Pack:** רדיו מיקי **#2** (4.8 / 49 ביקורות) | דודיק #1 (4.9 / 110) | אבנר #3 (4.6 / 67)
**תוצאות אורגניות:** דף Facebook של רדיו מיקי מדרג ראשון + מסגרות אחרות
**דף multimedia.html חדש:** נוצר וצפוי להיאינדקס ולשפר את הדירוג האורגני

### 3. מערכות איתור לרכב בטבריה

**Local Pack:** אלירן #1 | דודיק #2 | אבנר #3
**רדיו מיקי:** **אורגני #3** בחיפוש זה — תוצאה מצויינת!
**קישור באתר:** easy.co.il > מיגון ובטיחות > מכשירי איתור

### 4. חילוץ קודניות בטבריה

**AI Overview פעיל!** גוגל מציג תשובות AI עם רשימת נעילני קודן ומחלפי קודניות
**Local Pack:** נעילני קודן ומנעולים — לא חנות אביזורי רכב
**רדיו מיקי:** לא מופיע עדיין — **הזדמנות גדולה!** אין תחרות מחנות אביזורי רכב בטופ Local Pack
**פועל נדרש:** דף נישא קודנית יכול להכניס את רדיו מיקי ל-AI Overview
